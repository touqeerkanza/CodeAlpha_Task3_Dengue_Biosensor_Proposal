# CodeAlpha Biotechnology Internship — Task 4: Mini-Project Proposal

## 🩺 Low-Cost Paper-Based Electrochemical Biosensor for Rapid Dengue NS1 Antigen Detection

A mini-project proposal prepared for **Task 4** of the CodeAlpha Biotechnology Virtual Internship.

## 📄 About

A 5-page project proposal for a low-cost, paper-based electrochemical biosensor designed to detect the dengue NS1 antigen — the earliest reliable biomarker of active dengue infection. The project aims to close the gap between expensive, accurate lab-based ELISA testing and cheap but less sensitive rapid antigen strips, particularly for low-resource clinics in dengue-endemic regions.

The proposal includes:

- **Introduction** — dengue burden, the NS1 biomarker, and the diagnostic gap it targets
- **Problem Statement** — uneven access to accurate diagnostics in low-resource healthcare settings
- **Objectives** — five measurable project goals, from electrode design to cost analysis
- **Methodology** — a 5-phase, ~10–12 week plan: electrode fabrication, biorecognition layer immobilization, sensitivity/specificity testing, prototype integration, and cost analysis
- **Expected Outcomes** — a working prototype, reproducible protocol, and cost/performance comparison against ELISA and rapid strips
- **References** — grounded in recent peer-reviewed electrochemical biosensor research
